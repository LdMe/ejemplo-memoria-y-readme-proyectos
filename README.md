# Ejemplo de memoria y Readme de un proyecto

En la carpeta `docs` encontrarás:
- `README.md`: Archivo de ejemplo con instrucciones para ejecutar el proyecto.
- `MEMORIA.pdf`: Un documento de ejemplo detallado que explica la arquitectura, decisiones de diseño y funcionalidades implementadas.
- `guía documentación.pdf`: Un manual para crear documentación técnica y README de proyectos.
- `diagramas.pdf`: Breve guía con diferentes tipos de diagramas UML y su uso en proyectos de software.
